# Topic Breakdown

**1.** What do I need to know before deep-diving into this topic?\
**2.** Why does this topic exist? What problem happened that caused this topic to be created? (Keep it simple and short.)\
**3.** What features does this topic offer?\
**4.** How does this topic work in the big picture?\
**5.** What minimum data or input does the topic need so it can operate?\
**6.** What are the critical things I must know about this topic?\
**7.** What things can we do with this topic? Can it be used for A, B, or C?\
**8.** What other topics are usually used or connected with this topic?\
**9.** What advanced concepts or features exist in this topic?\
**10.** What are the common misconceptions about this topic?\
**11**. Where do i go next after learning this<br>
